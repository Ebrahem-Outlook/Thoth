# Continuous Learning Source Policy

Enabled sources must include source ID, type, official location, license identifier, license reference, attribution requirement, allowed content types, rate limit, route, trust level, and legal review date.

Enabled now:

- `owned-curriculum`: local deterministic Thoth-owned examples.
- `mdn-content-readme`: one allowlisted official MDN GitHub raw document, bounded to 256 KB per cycle, license-tracked as `CC-BY-SA-2.5`.

Disabled until further review:

- Wikimedia incremental sources.
- Allowlisted code repositories.
- Generic crawler.

No private repositories, credentials, sessions, paywalled content, arbitrary websites, or unreviewed raw internet content are allowed.
